#### 运行项目

**安装**

```bash
$ cnpm install mobx -S
$ cnpm install mobx-react -S
$ cnpm install @babel/plugin-proposal-decorators -D
$ cnpm install @babel/plugin-proposal-class-properties -D

$ cnpm install  customize-cra react-app-rewired --save

$ cnpm install antd --save

$ cnpm install less --save
$ cnpm install less-loader@5.0.0 --save

$ cnpm install babel-plugin-import --save
```

**运行**

```bash
$ npm init
$ npm start
```
